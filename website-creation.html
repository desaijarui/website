Website Creation

Kentucky Horsemen’s Benevolent and Protective Association’s Website:

This website development project was completed by my group in my CIS 420 class. This project involved the development of a website and simple content management system for KYHBPA. KYHBPA is a non-profit organization that would like to revamp their website, so it can best fit their needs to better achieve their future goals. In order to complete this project our group, which consisted of seven individuals including myself, used .NET as well as Visual Studio in order to make our project plan a reality. The following are a few screenshots of the website that we had created for the organization along with the GitHub link to all the website details.

The Indian Student Association’s Website:

 The Indian Student Association is a student organization on UofL’s campus that I have been a part of since my freshman year. Throughout my time in the club, I helped create and maintain two different websites. The first is the main website and the second includes a website for the dance competition that will be hosted in the spring of 2018. Below are a few screenshots of the two websites as well as links to them.

ISA’s Main Website:

This website is there to inform the community and campus about the Indian Student Association and the events that we host each year. This website is the main source of contact to get access to meeting minutes, event details, ticket information, and more. We have used Squarespace in order to create the basic layout of the website and used HTML and CSS to customize it to best fit out needs. View full website here.

Derby City Dhoom’s Website:

This website was created in order to inform bhangra and fusion teams and universities in the country about the dance competition that UofL’s ISA will be hosting spring of 2018. This website is critical to the success of the event because all application and deadline information is listed here first. It also helps for promotion reasons so that we can spread the word to the community and colleges as quickly as possible. In order to complete this website, we used Wix, a website generator, in order to get this project started. Additionally, HTML and CSS has been used to change the look and feel of the website so it is best catered to our needs. View full website here.
